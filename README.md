# API Documentation

Добро пожаловать в документацию нашего API!

## Быстрый старт

1. Получите API ключ
2. Установите базовый URL: `https://boost.simplebot.cfd/api/`
3. Начните делать запросы!

## Основные endpoints

- **Аутентификация** - `/auth`
- **Пользователи** - `/users`
- **Продукты** - `/products`
- **Заказы** - `/orders`

## Пример запроса

```bash
curl -X GET "https://api.example.com/v1/users" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
